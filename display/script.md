# CryptoFlow 视频脚本

**标题**: 【南开大学26C++】CryptoFlow — 交互式密码学原理可视化演示系统  
**目标时长**: 5-8 分钟  
**格式**: 纯演示 + 字幕解说（无声音）  
**输出**: MP4 (H.264, 1280x720)

---

## 自动化工作流

### 架构

```
record.sh → screencapture 录制 → fragments/ → compose.sh → output.mp4
                                                            ↑
                                              subtitles.srt (字幕)
```

### 工具依赖

```bash
# macOS 自带
screencapture    # 录屏
osascript        # AppleScript 控制
ffmpeg           # 视频合成（需安装：brew install ffmpeg）
```

### 快速开始

```bash
cd docs/vedios/

# 1. 录制所有片段
./record.sh

# 2. 合成最终视频
./compose.sh

# 输出: output/CryptoFlow_demo.mp4
```

## 手动录制流程

如果不用自动化脚本，按以下步骤操作：

### 1. 准备工作
```bash
# 编译项目
cmake --build build

# 创建目录
mkdir -p docs/vedios/fragments docs/vedios/output
```

### 2. 录制每个片段
```bash
# 打开 QuickTime Player → 新建屏幕录制
# 选择 CryptoFlow 窗口区域
# 按脚本操作每个算法
# 录制完成后导出为 MOV 格式
# 重命名为对应片段名称
```

### 3. 合成视频
```bash
# 使用 ffmpeg 合成
cd docs/vedios

# 拼接所有片段
ffmpeg -f concat -safe 0 -i filelist.txt \
  -c:v libx264 -pix_fmt yuv420p \
  -vf "subtitles=subtitles.srt:force_style='FontSize=24'" \
  output/CryptoFlow_demo.mp4
```
