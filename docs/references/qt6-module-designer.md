# Qt Widgets Designer C++ Classes

Provides classes to create your own custom widget plugins for Qt Widgets Designer and classes to access Qt Widgets Designer components.

## Classes

  [QAbstractExtensionFactory](qabstractextensionfactory.html)                                 Interface for extension factories in Qt Widgets Designer
  [QAbstractExtensionManager](qabstractextensionmanager.html)                                 Interface for extension managers in Qt Widgets Designer
  [QAbstractFormBuilder](qabstractformbuilder.html)                                           Default implementation for classes that create user interfaces at run-time
  [QDesignerActionEditorInterface](qdesigneractioneditorinterface.html)                       Allows you to change the focus of Qt Widgets Designer's action editor
  [QDesignerContainerExtension](qdesignercontainerextension.html)                             Allows you to add pages to a custom multi-page container in Qt Widgets Designer's workspace
  [QDesignerCustomWidgetCollectionInterface](qdesignercustomwidgetcollectioninterface.html)   Allows you to include several custom widgets in one single library
  [QDesignerCustomWidgetInterface](qdesignercustomwidgetinterface.html)                       Enables Qt Widgets Designer to access and construct custom widgets
  [QDesignerDynamicPropertySheetExtension](qdesignerdynamicpropertysheetextension.html)       Allows you to manipulate a widget's dynamic properties in Qt Widgets Designer's property editor
  [QDesignerFormEditorInterface](qdesignerformeditorinterface.html)                           Allows you to access Qt Widgets Designer's various components
  [QDesignerFormWindowCursorInterface](qdesignerformwindowcursorinterface.html)               Allows you to query and modify a form window's widget selection, and in addition modify the properties of all the form's widgets
  [QDesignerFormWindowInterface](qdesignerformwindowinterface.html)                           Allows you to query and manipulate form windows appearing in Qt Widgets Designer's workspace
  [QDesignerFormWindowManagerInterface](qdesignerformwindowmanagerinterface.html)             Allows you to manipulate the collection of form windows in Qt Widgets Designer, and control Qt Widgets Designer's form editing actions
  [QDesignerMemberSheetExtension](qdesignermembersheetextension.html)                         Allows you to manipulate a widget's member functions which is displayed when configuring connections using Qt Widgets Designer's mode for editing signals and slots
  [QDesignerObjectInspectorInterface](qdesignerobjectinspectorinterface.html)                 Allows you to change the focus of Qt Widgets Designer's object inspector
  [QDesignerPropertyEditorInterface](qdesignerpropertyeditorinterface.html)                   Allows you to query and manipulate the current state of Qt Widgets Designer's property editor
  [QDesignerPropertySheetExtension](qdesignerpropertysheetextension.html)                     Allows you to manipulate a widget's properties which is displayed in Qt Designer's property editor
  [QDesignerTaskMenuExtension](qdesignertaskmenuextension.html)                               Allows you to add custom menu entries to Qt Widgets Designer's task menu
  [QDesignerWidgetBoxInterface](qdesignerwidgetboxinterface.html)                             Allows you to control the contents of Qt Widgets Designer's widget box
  [QExtensionFactory](qextensionfactory.html)                                                 Allows you to create a factory that is able to make instances of custom extensions in Qt Designer
  [QExtensionManager](qextensionmanager.html)                                                 Extension management facilities for Qt Widgets Designer
  [QFormBuilder](qformbuilder.html)                                                           Used to dynamically construct user interfaces from UI files at run-time

## Detailed Description

In addition, the [QFormBuilder](qformbuilder.html) class provides the possibility of constructing user interfaces from UI files at run-time.
