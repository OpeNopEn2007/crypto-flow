# Qt Remote Objects C++ Classes

Provides an easy to use mechanism for sharing a [QObject](qobject.html)'s properties, signals, or slots, between processes. [More\...](#details)

## Namespaces

  [QtRemoteObjects](qtremoteobjects.html)   Contains identifiers used in the Remote Objects module, as well as some functions used from code generated by the Replica Compiler (repc)

## Classes

  [QAbstractItemModelReplica](qabstractitemmodelreplica.html)                       Serves as a convenience class for Replicas of Sources based on QAbstractItemModel
  [QRemoteObjectAbstractPersistedStore](qremoteobjectabstractpersistedstore.html)   A class which provides the methods for setting PROP values of a replica to value they had the last time the replica was used
  [QRemoteObjectDynamicReplica](qremoteobjectdynamicreplica.html)                   A dynamically instantiated Replica
  [QRemoteObjectHost](qremoteobjecthost.html)                                       A (Host) Node on a Qt Remote Objects network
  [QRemoteObjectHostBase](qremoteobjecthostbase.html)                               Base functionality common to Host and RegistryHost classes
  [QRemoteObjectNode](qremoteobjectnode.html)                                       A node on a Qt Remote Objects network
  [QRemoteObjectPendingCall](qremoteobjectpendingcall.html)                         Encapsulates the result of an asynchronous method call
  [QRemoteObjectPendingCallWatcher](qremoteobjectpendingcallwatcher.html)           Provides a QObject-based API for watching a QRemoteObjectPendingCall
  [QRemoteObjectPendingReply](qremoteobjectpendingreply.html)                       A templated version of QRemoteObjectPendingCall
  [QRemoteObjectRegistry](qremoteobjectregistry.html)                               A class holding information about Source objects available on the Qt Remote Objects network
  [QRemoteObjectRegistryHost](qremoteobjectregistryhost.html)                       A (Host/Registry) node on a Qt Remote Objects network
  [QRemoteObjectReplica](qremoteobjectreplica.html)                                 A class interacting with (but not implementing) a Qt API on the Remote Object network
  [QtROClientFactory](qtroclientfactory.html)                                       A class that holds information about client backends available on the Qt Remote Objects network
  [QtROServerFactory](qtroserverfactory.html)                                       A class that holds information about server backends available on the Qt Remote Objects network

## Detailed Description

To link against the module using CMake, add the following lines to your `cmake` file:

``` cpp
find_package(Qt6 REQUIRED COMPONENTS RemoteObjects)
target_link_libraries(directconnectserver PRIVATE Qt6::RemoteObjects)
```

To link against the module using `qmake`, add this line to your `.pro` file:

``` cpp
QT       += remoteobjects
```

For more information, see [Getting Started with Qt Remote Objects](qtremoteobjects-gettingstarted.html).
