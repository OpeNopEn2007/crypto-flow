# Qt SVG C++ Classes

The Qt SVG module provides functionality for handling SVG images. [More\...](#details)

## Namespaces

  [QtSvg](qtsvg.html)   Contains types used by the svg classes

## Classes

  [QSvgGenerator](qsvggenerator.html)   Paint device that is used to create SVG drawings
  [QSvgRenderer](qsvgrenderer.html)     Used to draw the contents of SVG files onto paint devices

## Detailed Description

Scalable Vector Graphics (SVG) is an XML-based language for describing two-dimensional vector graphics. Qt provides classes for rendering and displaying SVG drawings in widgets and on other paint devices. See the class descriptions for further details.

The [Qt SVG Widgets](qtsvgwidgets-module.html) library provides C++ classes for rendering SVG drawings into widget UIs.

  [QGraphicsSvgItem](qgraphicssvgitem.html)   QGraphicsItem that can be used to render the contents of SVG files
  [QSvgWidget](qsvgwidget.html)               Widget that is used to display the contents of Scalable Vector Graphics (SVG) files
