# Qt CoAP C++ Classes

Provides classes to use CoAP protocol. [More\...](#details)

## Namespaces

  [QtCoap](qtcoap.html)   Contains miscellaneous identifiers used throughout the Qt CoAP module

## Classes

  [QCoapClient](qcoapclient.html)                                   Allows the application to send CoAP requests and receive replies
  [QCoapMessage](qcoapmessage.html)                                 Holds information about a CoAP message that can be a request or a reply
  [QCoapOption](qcoapoption.html)                                   Holds data about CoAP options
  [QCoapPrivateKey](qcoapprivatekey.html)                           Interface for managing CoAP security keys
  [QCoapReply](qcoapreply.html)                                     Holds the data of a CoAP reply
  [QCoapRequest](qcoaprequest.html)                                 Holds a CoAP request. This request can be sent with QCoapClient
  [QCoapResource](qcoapresource.html)                               Holds information about a discovered resource
  [QCoapResourceDiscoveryReply](qcoapresourcediscoveryreply.html)   Holds the data of a CoAP reply for a resource discovery request
  [QCoapSecurityConfiguration](qcoapsecurityconfiguration.html)     Holds configuration options during the authentication process
