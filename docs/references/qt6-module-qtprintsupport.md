# Qt Print Support C++ Classes

The Qt PrintSupport module provides classes to make printing easier and portable.

## Classes

  [QAbstractPrintDialog](qabstractprintdialog.html)   Base implementation for print dialogs used to configure printers
  [QPageSetupDialog](qpagesetupdialog.html)           Configuration dialog for the page-related options on a printer
  [QPrintDialog](qprintdialog.html)                   Dialog for specifying the printer's configuration
  [QPrintEngine](qprintengine.html)                   Defines an interface for how QPrinter interacts with a given printing subsystem
  [QPrintPreviewDialog](qprintpreviewdialog.html)     Dialog for previewing and configuring page layouts for printer output
  [QPrintPreviewWidget](qprintpreviewwidget.html)     Widget for previewing page layouts for printer output
  [QPrinter](qprinter.html)                           Paint device that paints on a printer
  [QPrinterInfo](qprinterinfo.html)                   Gives access to information about existing printers

## Detailed Description

The [Qt Print Support](qtprintsupport-index.html) page contains information about using the module.

Qt Print Support is not available on iOS.
