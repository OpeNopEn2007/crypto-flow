## Detailed Description

The [Qt OpenGL](qtopengl-index.html) page contains information about how to use the module.

A separate [Qt OpenGL Widgets](qtopenglwidgets-module.html) library provides a C++ widget class for rendering OpenGL graphics into widgets UIs.

  [QOpenGLWidget](qopenglwidget.html)   Widget for rendering OpenGL graphics
