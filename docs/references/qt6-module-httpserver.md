# Qt HTTP Server C++ Classes

List of C++ classes that provide HTTP server framework. [More\...](#details)

This module was introduced in Qt 6.4.

## Classes

  [QAbstractHttpServer](qabstracthttpserver.html)                                   API to subclass to implement an HTTP server
  [QHttpServer](qhttpserver.html)                                                   Simplified API for QAbstractHttpServer and QHttpServerRouter
  [QHttpServerConfiguration](qhttpserverconfiguration.html)                         Controls server parameters
  [QHttpServerRequest](qhttpserverrequest.html)                                     Encapsulates an HTTP request
  [QHttpServerResponder](qhttpserverresponder.html)                                 API for sending replies from an HTTP server
  [QHttpServerResponse](qhttpserverresponse.html)                                   Encapsulates an HTTP response
  [QHttpServerRouter](qhttpserverrouter.html)                                       Provides functions to bind a path to a ViewHandler
  [QHttpServerRouterRule](qhttpserverrouterrule.html)                               The base class for QHttpServerRouter rules
  [QHttpServerWebSocketUpgradeResponse](qhttpserverwebsocketupgraderesponse.html)   Response to return when verifying WebSocket upgrades on HTTP server
