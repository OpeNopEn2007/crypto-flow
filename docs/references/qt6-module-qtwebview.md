# Qt WebView

> **来源**: [https://doc.qt.io/qt-6/qtwebview-module.html](https://doc.qt.io/qt-6/qtwebview-module.html)
> **获取日期**: 2026-05-12
> **描述**: WebView 组件，提供在 QML 和 C++ 中显示 Web 内容的原生 WebView

## Namespaces

  [QtWebView](qtwebview.html)   Functions that makes it easier to set-up and use the WebView

## Classes

  [QWebView](qwebview.html)                         Window that is used to view web content
  [QWebViewLoadingInfo](qwebviewloadinginfo.html)   Load status information for the QWebView::loadingChanged() signal
  [QWebViewSettings](qwebviewsettings.html)         Allows configuration of browser properties and attributes

## Detailed Description

The Qt [WebView](qml-qtwebview-webview.html) module provides functionality to initialize the backend for using the [WebView](qml-qtwebview-webview.html) QML type.

To include the module's definitions, use the following directive:

```
#include <QtWebView>
```
