# Qt WebChannel C++ Classes

List of C++ classes that provide the Qt WebChannel functionality. [More\...](#details)

## Classes

  [QWebChannel](qwebchannel.html)                                     Exposes QObjects to remote HTML clients
  [QWebChannelAbstractTransport](qwebchannelabstracttransport.html)   Communication channel between the C++ QWebChannel server and a HTML/JS client

## Detailed Description

To use these classes in your application, use the following include statement:

```
#include <QtWebChannel/QtWebChannel>
```

To link against the module, add this line to your [qmake](qmake-manual.html) `.pro` file:

```
QT += webchannel
```
