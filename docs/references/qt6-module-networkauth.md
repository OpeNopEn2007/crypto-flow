# Qt Network Authorization C++ Classes

Provides classes for network authorization support (OAuth). [More\...](#details)

## Classes

  [QAbstractOAuth](qabstractoauth.html)                                   The base of all implementations of OAuth authentication methods
  [QAbstractOAuth2](qabstractoauth2.html)                                 The base of all implementations of OAuth 2 authentication methods
  [QAbstractOAuthReplyHandler](qabstractoauthreplyhandler.html)           Handles replies to OAuth authentication requests
  [QOAuth1](qoauth1.html)                                                 Implementation of the OAuth 1 Protocol
  [QOAuth1Signature](qoauth1signature.html)                               Implements OAuth 1 signature methods
  [QOAuth2AuthorizationCodeFlow](qoauth2authorizationcodeflow.html)       Implementation of the Authorization Code Grant flow
  [QOAuth2DeviceAuthorizationFlow](qoauth2deviceauthorizationflow.html)   Implementation of the Device Authorization Grant flow
  [QOAuthHttpServerReplyHandler](qoauthhttpserverreplyhandler.html)       Handles loopback redirects by setting up a local HTTP server
  [QOAuthUriSchemeReplyHandler](qoauthurischemereplyhandler.html)         Handles private/custom and https URI scheme redirects
