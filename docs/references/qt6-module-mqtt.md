# Qt MQTT C++ Classes

Provides classes that enable sending messages via the MQTT protocol. [More\...](#details)

## Namespaces

  [QMqtt](qmqtt.html)   Contains miscellaneous identifiers used throughout the Qt MQTT module

## Classes

  [QMqttAuthenticationProperties](qmqttauthenticationproperties.html)       Represents configuration options during the authentication process
  [QMqttClient](qmqttclient.html)                                           Represents the central access communicating with an MQTT broker
  [QMqttConnectionProperties](qmqttconnectionproperties.html)               Represents configuration options a QMqttClient can pass to the server when invoking QMqttClient::connectToHost()
  [QMqttLastWillProperties](qmqttlastwillproperties.html)                   Represents configuration options a QMqttClient can pass to the server when specifying the last will during connecting to a server
  [QMqttMessage](qmqttmessage.html)                                         Information about a message received from a message broker based on a subscription
  [QMqttMessageStatusProperties](qmqttmessagestatusproperties.html)         Represents additional information provided by the server during message delivery
  [QMqttPublishProperties](qmqttpublishproperties.html)                     Represents configuration options for sending or receiving a message
  [QMqttServerConnectionProperties](qmqttserverconnectionproperties.html)   Represents configuration options of a server a QMqttClient is connected to
  [QMqttStringPair](qmqttstringpair.html)                                   Represents the string pair data type of the MQTT 5.0 standard
  [QMqttSubscription](qmqttsubscription.html)                               Receives notifications from an MQTT broker about a specified topic
  [QMqttSubscriptionProperties](qmqttsubscriptionproperties.html)           Represents configuration options a QMqttClient can pass to the server when subscribing to a topic filter
  [QMqttTopicFilter](qmqtttopicfilter.html)                                 Represents a MQTT topic filter
  [QMqttTopicName](qmqtttopicname.html)                                     Represents a MQTT topic name
  [QMqttUnsubscriptionProperties](qmqttunsubscriptionproperties.html)       Represents configuration options a QMqttClient can pass to the server when unsubscribing from a topic filter
  [QMqttUserProperties](qmqttuserproperties.html)                           Represents a list of QMqttStringPair values which can be passed to the server
